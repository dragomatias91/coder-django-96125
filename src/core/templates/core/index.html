Django

The Web framework for perfectionists with deadlines

Django es un framework web de alto nivel, basado en Python, que fomenta el desarrollo rápido y un diseño limpio y pragmático.

Fue construido por desarrolladores experimentados para encargarse de gran parte de la configuración del desarrollo web, permitiéndote concentrarte en escribir tu aplicación sin necesidad de reinventar la rueda.

Ridículamente rápido

Diseñado para ayudar a los desarrolladores a llevar aplicaciones del concepto a la realidad lo más rápido posible.

Seguro por defecto

Ayuda a evitar errores comunes de seguridad como inyección SQL, XSS y CSRF.

Escalable

Capaz de manejar los picos de tráfico más pesados, desde blogs pequeños hasta redes sociales masivas.

Baterías incluidas

Incluye docenas de extras para autenticación, administración de contenido, mapas y más.
